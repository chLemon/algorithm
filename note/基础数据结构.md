## zip

就像拉链一样，将多个可迭代对象（如列表、元组）的元素成对地打包在一起。

```python
names = ["Alice", "Bob", "Charlie"]
scores = [85, 92, 78]

# 打包
zipped = zip(names, scores)
print(list(zipped)) 
# 输出: [('Alice', 85), ('Bob', 92), ('Charlie', 78)]
```

你需要知道的 3 个关键特性

- 长度对齐（木桶效应）：
  如果传入的序列长度不一致，zip 会以最短的那个为准，多出来的部分会被直接丢弃。注意： 如果你希望以最长的为准并填充缺失值，可以使用 itertools.zip_longest()。
- 节省内存：
  在 Python 3 中，zip() 返回的是一个迭代器（Lazy Map）。它不会立即在内存中生成完整的列表，只有在你遍历它时才会产生数据。
- 解压（Unzip）：
  你可以利用 * 操作符（解包）来完成 zip 的逆向操作。

```Python
pairs = [('Alice', 85), ('Bob', 92)]
names, scores = zip(*pairs)
# names -> ('Alice', 'Bob')
```

常见使用场景

同时遍历多个列表
`for name, score in zip(names, scores):`

快速构建字典
`dict(zip(keys, values))`

矩阵转置
`transpose = list(zip(*matrix))`

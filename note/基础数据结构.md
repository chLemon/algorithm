## zip

就像拉链一样，将多个可迭代对象（如列表、元组）的元素成对地打包在一起。

```python
names = ["Alice", "Bob", "Charlie"]
scores = [85, 92, 78]

# 打包
zipped = zip(names, scores)
print(list(zipped)) 
# 输出: [('Alice', 85), ('Bob', 92), ('Charlie', 78)]
```

你需要知道的 3 个关键特性

- 长度对齐（木桶效应）：
  如果传入的序列长度不一致，zip 会以最短的那个为准，多出来的部分会被直接丢弃。注意： 如果你希望以最长的为准并填充缺失值，可以使用 itertools.zip_longest()。
- 节省内存：
  在 Python 3 中，zip() 返回的是一个迭代器（Lazy Map）。它不会立即在内存中生成完整的列表，只有在你遍历它时才会产生数据。
- 解压（Unzip）：
  你可以利用 * 操作符（解包）来完成 zip 的逆向操作。

```Python
pairs = [('Alice', 85), ('Bob', 92)]
names, scores = zip(*pairs)
# names -> ('Alice', 'Bob')
```

常见使用场景

同时遍历多个列表
`for name, score in zip(names, scores):`

快速构建字典
`dict(zip(keys, values))`

矩阵转置
`transpose = list(zip(*matrix))`

## pairwise

在 Python 中，pairwise 是一个非常实用的工具，主要用于将一个序列中的元素“成对”地提取出来。

简单来说，如果你有一个序列 [A, B, C, D]，pairwise 会让你得到 (A, B), (B, C), (C, D)。这种操作在处理路径、计算相邻差值或对比连续数据时非常高效。

在 Python 3.10+ 中（最推荐）
pairwise 已经正式加入了标准库 itertools。

```Python
from itertools import pairwise

data = [1, 10, 100, 1000]
for a, b in pairwise(data):
    print(f"当前对: {a}, {b}")

# 输出:
# 当前对: 1, 10
# 当前对: 10, 100
# 当前对: 100, 1000
```

注意：如果输入序列少于 2 个元素，pairwise 会直接返回空，因为它凑不成一对。
